### Аналіз відкритих даних: Екологічна та радіаційна обстановка в зоні АЕС (Open Data AI Analytics)
Проект має на меті створити відтворюваний конвеєр аналізу даних для моніторингу екологічної ситуації навколо атомних електростанцій. Проєкт включає автоматизоване завантаження, перевірку якості, дослідження та візуалізацію даних, щоб виявити довгострокові тенденції та потенційні аномалії в радіаційному фоні.

**Джерело відкритих даних**
*   Екологічна та радіаційна обстановка в зоні розташування атомних електростанцій
*   [https://data.gov.ua/dataset/4a9d3d56-bd95-4c3e-97e7-1cdc7bcbd445](https://data.gov.ua/dataset/4a9d3d56-bd95-4c3e-97e7-1cdc7bcbd445)
*   Набір даних містить результати спостережень за потужністю експозиційної дози гамма-випромінювання, метеорологічні параметри та проби об'єктів навколишнього середовища в зонах спостереження АЕС.

**Питання для аналізу**
1.  Тенденції в викидах Cs-137 та Co-60 за останні роки.
2.  Порівняння середніх показників радіаційного фону між різними атомними електростанціями.
3.  Допустимі рівнів викидів та фактичні викиди.
